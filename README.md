# 📚 Study Planner

A simple Study Planner web application built using **Flask** and **SQLite** to help students manage their study tasks.

## 🚀 Live Demo

https://study-planner-web-app-x7ms.onrender.com

## ✨ Features

- Add Tasks
- Search Tasks
- Edit Tasks
- Complete Tasks
- Delete Tasks
- Dashboard (Total, Pending, Completed)

## 🛠 Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- Git
- GitHub
- Render
